Introduction¶

Dataset Description¶

This dataset, sourced from The Movie Database (TMDb), includes detailed information on 1,000 movies. The aim of this analysis is to uncover relationships and trends between various attributes—such as revenue, popularity, and other associated features—potentially driving the success of these films.

Dataset Columns¶

  • id: Unique identifier for each movie within the dataset.
  • imdb_id: The movie's IMDb identifier, linking to IMDb for additional information.
  • popularity: Popularity rating of the movie on TMDb.
  • budget: Initial production budget of the movie (in USD).
  • revenue: Total revenue generated by the movie (in USD).
  • original_title: Original title of the movie.
  • cast: Main cast members in the movie.
  • homepage: Official website or homepage of the movie, if available.
  • director: Director(s) of the movie.
  • tagline: Catchphrase or slogan associated with the movie.
  • keywords: Keywords or phrases associated with the movie's themes or content.
  • overview: Brief summary or synopsis of the movie's plot.
  • runtime: Duration of the movie in minutes.
  • genres: Genre(s) that the movie belongs to (e.g., Drama, Action).
  • production_companies: Companies involved in the production of the movie.
  • release_date: Official release date of the movie.
  • vote_count: Number of user reviews or votes on TMDb.
  • vote_average: Average rating given by users on TMDb.
  • release_year: Year the movie was released.
  • budget_adj: Budget of the movie adjusted for inflation to reflect current USD value.
  • revenue_adj: Revenue of the movie adjusted for inflation to reflect current USD value.

This dataset provides a comprehensive view of each movie’s financial and critical metrics, along with contextual information on cast, production, and thematic elements. Analyzing these variables will help identify factors that may influence movie success.

Function for Regression Chart Visualization¶

To analyze the relationships between various numerical variables in our dataset, we can define a function that creates a regression chart. This function will take in parameters for the x and y variables, allowing us to visualize the correlation between them through a scatter plot with a regression line. The correlation coefficient will be calculated and displayed on the plot, providing insight into the strength of the relationship.

The following function will enable us to generate these regression charts efficiently for any specified variables in our analysis:

In [12]:
# Import required libraries
import matplotlib.pyplot as plt
import seaborn as sns
from scipy.stats import linregress

# Define the function for drawing regression charts
def draw_regression_chart(x, y, data):
    """
    Draws a regression chart with a scatter plot and a regression line between two variables.
    Calculates and returns the correlation coefficient.

    Parameters:
    x (str): The column name for the x-axis variable.
    y (str): The column name for the y-axis variable.
    data (pd.DataFrame): The data frame containing the data.

    Returns:
    float: The correlation coefficient between x and y.
    """
    # Filter the DataFrame for non-zero values in both x and y columns
    filtered_data = data[(data[x] > 0) & (data[y] > 0)]

    # Set the plot size
    plt.figure(figsize=(12, 6))

    # Create a scatter plot with a regression line
    sns.regplot(x=x, y=y, data=filtered_data, scatter_kws={'alpha': 0.5}, line_kws={'color': 'red'})

    # Calculate the correlation coefficient
    correlation = filtered_data[x].corr(filtered_data[y])

    # Set plot title and labels
    plt.title(f'Relationship between Movie {x.capitalize()} and {y.capitalize()}', fontsize=16)
    plt.xlabel(f'{x.capitalize()} (in appropriate units)', fontsize=14)
    plt.ylabel(f'{y.capitalize()} (in dollars)', fontsize=14)

    # Display the correlation score on the plot
    plt.annotate(f'Correlation: {correlation:.2f}', xy=(0.05, 0.95), xycoords='axes fraction', fontsize=12, color='black')

    # Show the plot
    plt.grid()
    plt.show()

    return correlation
